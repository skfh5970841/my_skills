# reference.md — 채사장 스타일 문장 패턴 레퍼런스

각 패턴은 규칙이 아니라 채사장이 자주 선택하는 방식이다.
패턴을 기계적으로 적용하지 말고, 이 패턴이 왜 작동하는지를 파악해 판단에 활용한다.

---

## 문장 리듬 패턴

### P-RHYTHM-A: 단문 연속 → 장문 전환

**작동 원리**:
짧은 문장들은 독자의 호흡을 빠르게 만든다. 그 긴장이 쌓인 뒤 긴 문장이 오면,
독자는 자연스럽게 속도를 늦추고 개념을 받아들인다.
짧은 문장의 개수는 긴장이 충분히 쌓이는 지점에서 결정한다.

**예시 A**:
```
역사는 반복된다. 아니, 정확히 말하면 구조가 반복된다.
사람은 바뀌고, 시대는 바뀌고, 이름도 바뀌지만,
그 안에서 작동하는 힘의 방향은 놀랍도록 일정하다.
```

**예시 B**:
```
돈은 종이다. 그 이상도 이하도 아니다.
그런데 그 종이 한 장이 누군가의 하루를 바꾸고, 누군가의 인생을 끝내기도 한다.
우리가 돈에 부여한 의미가 돈 자체보다 훨씬 강력하기 때문이다.
```

**예시 C**:
```
자유는 좋은 것이다. 누구나 원한다.
문제는 자유가 하나가 아니라는 것이다.
어떤 자유는 다른 사람의 자유를 빼앗는 방식으로만 실현된다.
```

---

### P-RHYTHM-B: 반복 → 수렴

**작동 원리**:
같은 구조의 문장을 반복하면 독자는 리듬을 느끼고 패턴을 예상한다.
그 예상이 충족될 때 정리 문장이 힘을 얻는다.
반복 횟수는 숫자로 정하지 않는다. 리듬이 완성되면 멈춘다.

**예시 A** (3회):
```
봉건제는 토지가 권력이었던 시대다.
자본주의는 돈이 권력인 시대다.
지금 우리가 사는 시대는 정보가 권력이 되어가는 시대다.
무엇이 권력인지가 바뀌면, 세상이 바뀐다.
```

**예시 B** (3회):
```
마르크스는 경제가 역사를 만든다고 했다.
다윈은 환경이 생명을 만든다고 했다.
프로이트는 무의식이 인간을 만든다고 했다.
셋 다 같은 말을 하고 있다. 우리가 의식하지 못하는 무언가가 우리를 움직인다는 것이다.
```

**예시 C** (2회로 충분한 경우):
```
어떤 사람은 열심히 하면 된다고 말한다.
어떤 사람은 좋은 환경이 필요하다고 말한다.
두 주장 모두 맞다. 그리고 두 주장 모두 절반만 맞다.
```

→ 예시 C는 2회 반복만으로 수렴했다. 리듬이 완성됐으면 굳이 3회를 채우지 않는다.

---

### P-RHYTHM-C: 역접 단독 배치

**작동 원리**:
"그런데"를 단락 전환점에 두면 독자의 안도감을 깨고 다시 집중시킨다.
짧게 배치할수록 전환의 무게가 커진다.

**예시 A**:
```
민주주의는 가장 많은 사람이 선택한 것을 따르는 제도다. 논리적으로 완벽해 보인다.
그런데.
다수가 틀릴 수도 있다면 어떻게 되는가.
```

**예시 B**:
```
자본주의는 지난 200년간 인류를 빠르게 풍요롭게 만들었다. 이것은 부정하기 어렵다.
하지만 그 풍요가 모두에게 균등하게 닿았는가는 전혀 다른 이야기다.
```

**예시 C**:
```
소비는 나쁜 것이 아니다. 경제를 움직이는 기본 동력이다.
그러나 소비가 정체성이 되는 순간, 우리는 무언가를 잃기 시작한다.
```

## 유튜브 내레이션 리듬 패턴

### V-RHYTHM-A: 한 문장 한 생각

**작동 원리**:
영상 대본은 독자가 되돌아가 읽을 수 없다.
따라서 한 문장에는 하나의 생각만 담는다.
긴 문장이 필요할 때도, 핵심 생각이 바뀌는 지점에서는 문장을 나눈다.

**나쁜 예시**:
```
실존주의는 인간이 본질에 앞서 실존하며 자신의 선택을 통해 정체성을 구성한다는 철학적 입장이다.
```

**좋은 예시**:
```
실존주의는 이렇게 묻는다.
인간에게 정해진 답이 있는가.
아니면 우리는 선택하면서 자신이 누구인지 만들어가는가.
쉽게 말하면, 인간은 완성품으로 태어나는 것이 아니다.
```

---

### V-RHYTHM-B: 설명 → 다시 말하기 → 자막 문장

**작동 원리**:
영상에서는 중요한 개념을 한 번만 말하면 약하다.
설명한 뒤, 더 쉬운 말로 다시 말하고, 마지막에 자막으로 뽑을 수 있는 문장으로 수렴한다.

**구조**:
```
[개념 설명]
다시 말하면, [쉬운 재설명].
결국 [자막으로 뽑을 수 있는 한 문장].
```

**예시**:
```
자유는 단순히 하고 싶은 것을 하는 상태가 아니다.
다시 말하면, 자유는 하기 싫은 것을 거부할 수 있는 힘이기도 하다.
결국 자유는 선택의 문제가 아니라, 거부할 수 있는 조건의 문제다.
```

---

### V-RHYTHM-C: 중간 정리 문장

**작동 원리**:
영상에서는 설명이 길어질수록 시청자가 흐름을 놓친다.
3~5문단마다 짧은 정리 문장을 넣어 지금 어디까지 왔는지 알려준다.

**예시**:
```
여기까지 정리하면 이렇다.
문제는 의지가 약한 것이 아니라, 선택을 유지할 구조가 없다는 것이다.
```

---

### V-RHYTHM-D: 쇼츠 추출 가능 문장

**작동 원리**:
좋은 긴 대본 안에는 짧게 잘라도 의미가 살아 있는 문장이 있다.
다만 쇼츠용 문장을 만들기 위해 전체 구조를 망치면 안 된다.
쇼츠 문장은 본문에서 자연스럽게 나온 결과여야 한다.

**예시**:
```
우리가 실패라고 부르는 것 중 일부는 사실 의지의 문제가 아니라 설계의 문제다.
```

```
자유는 선택지가 많아지는 것이 아니라, 선택하지 않을 힘을 갖는 것이다.
```

---

### V-RHYTHM-E: 화면 전환을 고려한 문장

**작동 원리**:
영상에서는 문장이 화면을 부른다.
모든 문장에 화면 지시를 넣을 필요는 없지만, 장면이 바뀌어야 하는 지점에서는 문장도 함께 전환되어야 한다.

**예시**:
```
이제 질문을 조금 넓혀보자.
이 문제는 한 사람의 의지 문제가 아니라, 우리가 시간을 관리하는 방식의 문제다.
```

## 구조 패턴

### P-STRUCTURE-A: 나선형 확장

개인 경험 → 사회 현상 → 역사 맥락 → 철학적 질문 → 독자 현재로 귀환.
동심원을 그리듯 넓혀가고, 반드시 독자의 현재로 돌아온다.

**적용 예시**:
```
"왜 이렇게 불안할까" (개인)
→ 현대인의 불안은 개인 문제가 아니다 (사회)
→ 산업화 이후 공동체가 해체되면서 시작됐다 (역사)
→ 실존주의는 이를 '던져진 존재의 조건'이라 불렀다 (철학)
→ 그렇다면 이 불안을 어떻게 대할 것인가 (독자 귀환)
```

---

### P-STRUCTURE-B: 이중 반전

통념을 한 번 뒤집고, 그 뒤집은 것도 다시 뒤집어 더 깊은 이해에 도달한다.

```
[통념 A]
→ [반전 1: 사실은 B다]
→ [반전 2: 더 정확하게는 A와 B를 포함한 C다]
```

**적용 예시**:
```
역사는 승자의 기록이다. (통념)
→ 그래서 역사는 편향돼 있고 믿기 어렵다. (반전 1)
→ 그런데 그 편향 자체를 읽는 것이 역사를 이해하는 방법이다. (반전 2)
```

---

## 어휘 및 비유 판단 기준

### 서술어 선택 원칙

피해야 할 것은 특정 서술어가 아니라,
**글쓴이의 감정·판단이 독자 위에 군림하는 구조**다.

| 구조 | 판단 | 이유 |
|---|---|---|
| "나는 이것이 중요하다고 생각한다" | 피한다 | 글쓴이 판단을 강요한다 |
| "이것이 중요한 이유는 ~이기 때문이다" | 권장 | 구조로 설명한다 |
| "당신은 아마 이렇게 생각할 것이다" | 허용 | 독자 시점을 대변한다 |
| "~인 셈이다 / ~라고 할 수 있다" | 권장 | 중립적 서술, 독자 판단 여지를 남긴다 |

### 비유 판단 기준

비유는 개념을 더 명확하게 만들 때만 쓴다.
서정적이냐 아니냐가 기준이 아니라, **기능하느냐 아니냐**가 기준이다.

- 비유를 읽은 독자가 개념을 더 잘 이해하는가 → 쓴다
- 비유 자체는 아름답지만 개념과 따로 노는가 → 버린다

**기능하는 비유 예시**:
```
"규칙을 만드는 사람과 규칙 안에서 노는 사람"
→ 권력의 층위를 명확히 이미지화한다.

"빙산의 수면 아래처럼"
→ 보이지 않는 구조가 있다는 것을 즉시 시각화한다.
```

**기능하지 않는 비유 예시**:
```
"강물처럼 흘러가는 시간"
→ 시간이 흐른다는 것 외에 아무것도 추가하지 않는다.

"마치 첫사랑처럼 설레는 지식"
→ 지식의 속성을 설명하지 않는다. 감성만 추가한다.
```

---

### 전환어 목록

| 전환어 | 용도 | 사용 빈도 |
|---|---|---|
| 그런데 | 역접, 방향 전환 | 높음 — 단, 남발하면 효과가 줄어든다 |
| 흥미로운 것은 | 새로운 관점 도입 | 중간 |
| 사실 | 통념 수정, 재정의 | 중간 |
| 결국 | 수렴, 마무리 | 중간 |
| 다시 말하면 | 재설명 | 낮음 |

"그런데"는 채사장 스타일의 핵심 전환어지만, 한 단락 안에 두 번 이상 쓰면
역접의 무게가 사라진다. 한 단락에 한 번을 기본값으로 한다.

## 종합 적용 예시

**주제: 세금이란 무엇인가**

사용된 패턴: T-INTRO-A(C형) → T-CONCEPT-A → P-RHYTHM-B → T-INSIGHT-A

```
우리는 매달 세금을 낸다.
그런데 '세금이 무엇인가'라는 질문에 제대로 답할 수 있는 사람은 많지 않다.

세금은 단순히 국가가 가져가는 돈이 아니다.
우리가 사회를 유지하기 위해 합의한 비용이다.
문제는, 그 합의가 언제 어떻게 이루어졌는지를 기억하는 사람이 없다는 것이다.

역사적으로 세금은 권력의 언어였다.
왕은 세금을 통해 자신이 누구의 위에 있는지를 증명했다.
그런데 근대 이후 민주주의가 확산되면서 세금의 의미가 달라지기 시작했다.
권력의 언어에서 시민의 언어로.

세금을 '빼앗기는 것'으로 보는 사람은 사회를 개인들의 집합으로 본다.
세금을 '함께 내는 비용'으로 보는 사람은 사회를 공동체로 본다.
이 두 시각 사이의 갈등이 현대 정치의 본질이다.

처음 질문으로 돌아가보자. 세금이란 무엇인가.
이제 이 질문은 납세의 의무에 관한 것이 아니다.
우리가 어떤 사회에서 살고 싶은가에 관한 질문이다.
```
