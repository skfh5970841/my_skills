# template.md — 채사장 스타일 글쓰기 템플릿

각 템플릿은 공식이 아니라 **작동 원리 + 다양한 예시**다.
예시를 복사하지 말고, 원리를 이해해서 주제에 맞게 변형한다.

---

## 도입 템플릿

### T-INTRO-A: 질문형

**작동 원리**:
독자가 이미 하고 있는 행위 안에 숨어 있는 질문을 꺼낸다.
질문이 독자 자신의 것이어야 작동한다.
"세금이란 무엇인가"는 세금을 내는 사람의 질문이 될 수 있다.
"헤겔의 변증법이란 무엇인가"는 아직 독자의 질문이 아니다.

**적용 조건**: 독자가 그 행위나 상황을 실제로 경험하고 있을 때.

**예시 A — 일상 행위 포착형**:
```
우리는 매달 세금을 낸다.
그런데 '세금이 무엇인가'라는 질문에 제대로 답할 수 있는 사람은 많지 않다.
```

**예시 B — 경험 환기 후 방향 전환형**:
```
살면서 한 번쯤 '내 노력이 왜 결과로 이어지지 않을까'를 생각해본 적 있을 것이다.
이 질문은 보통 자기 자신을 향한다.
그런데 방향을 바꿔보면 어떨까.
```

**예시 C — 단문 직격형** (질문 없이도 가능):
```
역사는 반복된다.
이 말이 위안인지 경고인지는, 역사를 어떻게 읽느냐에 달려 있다.
```

→ 세 예시는 구조가 다르다. 공통점은 하나다. 독자를 질문 앞에 세운다.

---

### T-INTRO-B: 역설형

**작동 원리**:
독자가 당연하게 여기는 통념을 먼저 인정한 뒤 뒤집는다.
반전은 강할수록 좋다. 단, 뒤집을 통념이 독자 대부분이 실제로 믿는 것이어야 한다.

**적용 조건**: 글의 핵심이 기존 인식의 전복일 때.

**예시 A — 통념 → 반전 → 간극 강조형**:
```
우리는 민주주의가 다수결이라고 배웠다.
그런데 다수결은 민주주의의 여러 요소 중 하나일 뿐이다.
이 차이가 작아 보이지만, 여기서 민주주의 이해가 완전히 달라진다.
```

**예시 B — 인정 → 그러나형**:
```
자본주의가 지난 200년간 인류를 빠르게 풍요롭게 만든 것은 사실이다.
하지만 그 풍요가 모두에게 균등하게 닿았는가는 전혀 다른 이야기다.
```

**예시 C — 당연함 해체형**:
```
과학은 진리를 발견한다고 우리는 배웠다.
그런데 과학자들은 진리를 발견하는 것이 아니라, 더 나은 설명을 선택하는 것이라고 말한다.
```

---

### T-INTRO-C: 역사적 장면형

**작동 원리**:
독자를 구체적인 역사 장면 안으로 끌어들인다.
장면은 현재와 연결된다는 암시로 끝낸다. 연결을 직접 설명하지 않는다.

**적용 조건**: 역사 장면 자체가 질문형보다 강한 진입점일 때.
이 경우 첫 문장이 평서문이어도 그대로 둔다.

**예시 A — 붕괴 장면형**:
```
1929년, 뉴욕 월스트리트에서 주가가 폭락했다.
단 며칠 만에 수백만 명의 삶이 무너졌다.
이 사건은 경제 위기가 아니었다. 세계가 작동하는 방식에 대한 질문이었다.
```

**예시 B — 결정의 순간형**:
```
1789년, 프랑스 시민들은 왕을 끌어내렸다.
그들이 원한 것은 더 나은 왕이 아니었다.
왕 없는 세상을 상상하기 시작한 것이다.
```

**예시 C — 대화·선언형**:
```
1517년, 마르틴 루터는 교회 문에 95개의 반박문을 붙였다.
그는 교회를 무너뜨리려 한 것이 아니었다.
그러나 그 행동이 유럽을 반으로 갈랐다.
```

---

### T-INTRO-D: 개인 관찰형

**작동 원리**:
보편적인 개인 경험에서 출발해 철학적·사회적 구조로 확장한다.
독자가 "나도 그런 적 있다"고 느끼는 경험이어야 한다.

**적용 조건**: 개인 경험에서 사회 구조 또는 철학적 함의로 확장할 때.

**예시**:
```
살면서 한 번쯤은 내가 왜 이렇게 사는 걸까, 라는 생각을 해봤을 것이다.
이 질문을 불안의 증거라고 보는 사람도 있다.
철학자들은 이를 인간이 인간임을 증명하는 순간이라고 말한다.
```

## 개념 설명 템플릿

### T-CONCEPT-A: 추상→구체→비유→정리

**작동 원리**:
독자는 추상에서 출발하면 길을 잃는다.
구체적 사례로 내려온 뒤 비유로 이미지를 만들고, 한 문장으로 수렴한다.
비유는 개념을 더 명확하게 만들 때만 추가한다. 비유 자체가 목적이 아니다.

**기본 흐름**:
```
[추상적 정의, 1문장]
[역사적 또는 현실 사례, 2~4문장]
쉽게 말하면, [일상적 비유]   ← 개념이 시각화되지 않는다면 생략 가능
결국 [개념]은 [한 문장 정리].
```

**예시**:
```
잉여 가치란 노동자가 만들어낸 가치 중 임금으로 돌아오지 않는 부분이다.
하루에 100만 원어치 제품을 만드는 노동자가 30만 원을 받는다면,
나머지 70만 원은 자본가에게 귀속된다.
마르크스는 이 구조가 자본주의의 우연한 결함이 아니라 작동 원리 자체라고 봤다.
쉽게 말하면, 내가 일한 시간 중 나를 위한 시간과 타인을 위한 시간의 차이다.
결국 잉여 가치는 자본주의가 유지되는 연료다.
```

---

### T-CONCEPT-B: 점층적 심화

**작동 원리**:
독자가 이미 아는 설명을 먼저 인정하고, 그것이 불완전하다는 것을 보여준다.
"사실은 더 정확하게 말하면"의 구조로 깊이를 더한다.

**적용 조건**: 독자가 개념을 이미 어느 정도 알고 있을 때.

**흐름**:
```
[개념]은 [독자가 아는 설명]이라고 흔히 말한다.
사실 이것은 [개념]의 절반만 담은 설명이다.
더 정확하게 말하면, [더 깊은 설명]이다.
이 차이는 [실제 결과]에서 드러난다.
```

**예시**:
```
자유란 하고 싶은 것을 할 수 있는 상태라고 흔히 말한다.
사실 이것은 자유의 절반만 담은 설명이다.
더 정확하게 말하면, 자유는 하기 싫은 것을 거부할 수 있는 상태다.
이 차이는 우리가 어떤 사회 제도를 설계하느냐에서 결정적으로 드러난다.
```

---

### T-CONCEPT-C: 비교·대조

**작동 원리**:
두 개념을 나란히 놓아 차이를 드러낸다.
표면적 유사점을 먼저 인정하고, 결정적 차이로 이동한다.

**흐름**:
```
[A]는 [핵심 특징]이다.
[B]는 [핵심 특징]이다.
표면적으로 둘은 비슷해 보인다.
그런데 결정적인 차이는 [차이점]이다.
이 차이가 [실제 결과]를 만든다.
```

## 전개 템플릿

### T-BODY-A: 지식 연결 전개

**작동 원리**:
하나의 개념을 설명한 뒤, 전혀 다른 분야와의 연결로 독자가 예상하지 못한 통찰을 만든다.
연결은 억지로 만들지 않는다. 실제로 구조가 같을 때만 연결한다.

**흐름**:
```
[개념 설명]
흥미로운 것은 이 구조가 [전혀 다른 분야]와 정확히 같다는 점이다.
[다른 분야에서의 같은 구조 설명]
결국 [개념]은 [표면적 의미]가 아니라 [더 깊은 의미]다.
```

---

### T-BODY-B: 반복 강조형

**작동 원리**:
같은 구조의 문장을 반복해 패턴을 각인시키고, 정리 문장으로 수렴한다.
반복 횟수는 리듬이 완성된다고 느껴지는 지점에서 결정한다. 보통 2~4회.

**흐름**:
```
[사례 A]는 [공통 구조]였다.
[사례 B]는 [공통 구조]였다.
[사례 C]는 [공통 구조]였다.
[패턴을 한 문장으로 정리]
```

**주의**: 반복은 독자가 리듬을 느끼기 위한 것이다.
억지로 3개를 맞추는 것보다 2개가 더 강렬할 수 있다. 숫자보다 리듬을 본다.

## 통찰 도출 템플릿

### T-INSIGHT-A: 귀환형

**작동 원리**:
도입에서 던진 질문을 다시 꺼낸다.
이제 독자는 본문에서 쌓은 맥락으로 처음과 다른 답을 스스로 낼 수 있다.
글쓴이가 "이것이 답이다"라고 선언하지 않는다.

**흐름**:
```
처음으로 돌아가보자. [도입 질문 재환기]
이제 이 질문은 다르게 보인다.
[본문 맥락을 바탕으로 한 새로운 답, 2~3문장]
```

**예시**:
```
처음으로 돌아가보자. 세금이란 무엇인가.
이제 이 질문은 단순히 납세의 의무에 관한 것이 아니다.
우리가 어떤 사회에서 살고 싶은가에 관한 질문이다.
```

---

### T-INSIGHT-B: 역전형

**작동 원리**:
독자가 갖고 있던 인식을 명시적으로 뒤집는다.
"우리는 X라고 생각해왔다. 그런데 실제로는 Y다."

**예시**:
```
우리는 경제를 숫자의 학문이라고 생각해왔다.
그런데 경제는 인간의 욕망과 공포가 만들어내는 이야기다.
이것은 경제학의 이야기가 아니다.
우리가 왜 지금 이런 선택을 하는지를 이해하는 출발점이다.
```

---

### T-INSIGHT-C: 확장형

**작동 원리**:
개인 수준의 현상을 사회 구조, 역사 맥락으로 넓혀간다.
마지막 문장은 독자의 현재 현실로 돌아온다.

**흐름**:
```
이것은 개인의 문제가 아니다.
[사회 구조 차원 설명]
더 크게 보면, [역사·철학적 맥락]
결국 우리가 [경험]을 겪는 것은 [구조적 이유] 때문이다.
```

---

### T-INSIGHT-D: 열린 질문형

**작동 원리**:
마무리를 선언이나 정리가 아닌 의문문으로 닫는다.
닫히지 않은 질문은 독자의 내면으로 이전된다.
독자가 "나도 그렇다"고 스스로 느끼게 만드는 것이 목표다 — 글쓴이가 선언하면 안 된다.
이 형식은 개인적 통찰이나 존재론적 질문으로 귀결되는 글에 가장 유효하다.

**흐름**:
```
[글쓴이 자신의 경험이나 인식 변화를 1인칭으로 서술]
[그 경험이 가리키는 더 넓은 조건 또는 구조를 한 문장으로]
혹시 [독자 자신을 향한 의문문]?
```

**예시 A** (자기 인식):
```
나는 오랫동안 그 문제를 바깥에서 찾았다.
그런데 구조를 이해하고 나니, 내가 그 구조의 일부였다는 것이 보였다.
눈을 뜨고 있어도 보지 못하는 사람이 있다.
혹시 내가 그런 사람은 아니었을까.
```

**예시 B** (관계):
```
우리는 타인을 이해한다고 생각한다.
그런데 이해는 대개 자신의 언어로 상대를 번역한 것에 가깝다.
혹시 우리가 나눈 대화 중 진짜 상대의 말을 들은 순간이 얼마나 됐을까.
```

**판단 기준**:
- 의문문은 하나여야 한다. 두 개 이상이면 질문이 흩어진다.
- 독자를 향한 질문이어야 한다. 세계를 향한 질문은 다른 패턴이다.
- 이 형식을 남발하면 글이 항상 불안하게 끝난다는 인상을 준다.
  확실한 구조적 결론이 있을 때는 T-INSIGHT-A나 B를 쓴다.
