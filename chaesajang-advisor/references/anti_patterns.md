# anti_patterns.md — 채사장 스타일 금지 패턴

각 패턴은 "즉시 수정"이 아니라 **판단 기준 + 허용 예외**로 제시된다.
특정 표현이 문제가 아니라, 그 표현이 글 안에서 어떤 역할을 하는지가 기준이다.

---

## AP-01: 감성이 논리를 대체하는 패턴

**기준**: 감성 언어 앞에 구조적 설명이 있는가.
- 있다 → 허용. 감성은 논리의 결과로 자연스럽게 호출된다.
- 없다 → 수정. 감성이 설명을 건너뛰고 있다.

**수정 대상 예시** (감성이 논리를 대체):
```
역사는 우리에게 가슴 아픈 교훈을 남긴다.
수많은 사람들의 눈물과 희생 위에 지금 우리가 서 있다는 것을 잊지 말아야 한다.
```
→ 구조적 설명 없이 감성으로 결론을 낸다. 수정한다.

**수정 후**:
```
역사는 반복된다. 다시는 일어나지 않을 것이라는 맹세에도 불구하고.
같은 구조가 다시 작동한다. 그 구조를 보지 못하면 반복을 막을 수 없다.
```

**유지 가능한 예시** (논리 이후 감성 호출):
```
[앞에 구조적 설명이 충분히 쌓인 상태에서]
같은 전쟁이 같은 이유로 반복된 것이다. 희생자들의 이름이 달라졌을 뿐.
```
→ 마지막 문장의 감성은 앞의 논리가 만들어낸 것이다. 유지한다.

## AP-02: 자기계발 클리셰

**기준**: 독자에게 행동이나 태도를 명령·촉구하는가.
채사장의 글은 독자를 변화시키려 하지 않는다. 독자 스스로 생각하게 만든다.

**수정 대상 예시**:
```
지식은 당신의 삶을 바꿀 수 있다. 지금 당장 책을 펼쳐라. 아는 것이 힘이다.
```

**수정 후**:
```
지식은 세계를 다르게 보는 방법이다. 같은 뉴스를 보고 전혀 다른 것을 읽어낼 수 있다면,
그것이 지식의 효과다. 더 많이 아는 것이 아니라, 더 다르게 보는 것.
```

**허용 예외**: "~할 수 있다"는 가능성 제시는 클리셰가 아니다.
```
이 구조를 이해하면, 지금 일어나는 일들이 다르게 보일 수 있다.
```
→ 독자에게 명령하지 않고 가능성을 보여준다. 허용한다.

## AP-03: 근거 없는 단언

**기준**: 결론 앞에 독자가 납득할 수 있는 맥락과 설명이 충분히 쌓였는가.
- 쌓였다 → 강한 결론 문장도 허용된다.
- 쌓이지 않았다 → 결론을 앞에 두는 것은 단언이다. 수정한다.

**수정 대상 예시**:
```
결국 자본주의는 실패한 시스템이다.
```
→ 근거 없이 단언한다. 수정한다.

**수정 후**:
```
자본주의가 효율적이라는 것은 맞다. 하지만 효율적인 것이 항상 좋은 것과 같지는 않다.
이 구분에서 현대 사회의 많은 갈등이 시작된다.
```

**허용 예시** (맥락이 충분히 쌓인 후의 강한 결론):
```
[앞에 역사적 사례와 구조 설명이 충분히 쌓인 상태에서]
결국 이것은 제도의 실패가 아니라 설계의 문제였다.
```
→ 같은 단언 구조지만, 앞에서 충분히 쌓였다면 허용된다.

## AP-04: 전문 용어 번역 없이 나열

**기준**: 독자가 그 용어를 모를 가능성이 있는가.
독자 맥락상 이미 알고 있는 용어라면 번역 없이 써도 된다.

**수정 대상 예시**:
```
포스트모더니즘은 메타서사의 해체와 다원주의적 인식론을 기반으로 하며,
계몽주의적 이성 중심주의에 대한 비판적 전복을 시도한다.
```

**수정 후**:
```
포스트모더니즘은 간단히 말해, '절대적 진리는 없다'는 선언이다.
계몽주의가 이성으로 하나의 올바른 답을 찾을 수 있다고 믿었다면,
포스트모더니즘은 그 믿음 자체를 의심한다.
```

## AP-05: 판단 포기 양비론

**기준**: 두 입장을 나열한 뒤 실제로 아무 판단도 하지 않는가.
중립이 목표가 아니다. 두 입장이 추구하는 가치의 차이를 구조적으로 드러내는 것이 목표다.

**수정 대상 예시**:
```
자본주의에도 장단점이 있고, 사회주의에도 장단점이 있다.
어느 쪽이 더 좋은지는 사람마다 다르다.
```

**수정 후**:
```
자본주의와 사회주의를 비교하는 것은 잘못된 질문에서 출발한다.
두 시스템은 서로 다른 가치를 극대화하려 한다. 자본주의는 효율을, 사회주의는 평등을.
문제는 어느 쪽이 더 나은가가 아니라, 우리가 어떤 가치를 더 중요하게 여기는가다.
```

## AP-06: 논문·교과서체

**기준**: 문장이 능동적으로 움직이는가, 수동적으로 정의되는가.
채사장의 문장은 방향과 힘이 있다.

**수정 대상 예시**:
```
경제는 재화와 서비스의 생산, 분배, 소비에 관한 활동의 총체로 정의될 수 있으며,
이는 미시경제학과 거시경제학의 두 분야로 연구된다.
```

**수정 후**:
```
경제는 우리가 어떻게 먹고사는가에 대한 학문이다. 거창하게 들리지만 질문은 하나다.
누가 무엇을 얼마나 갖는가. 이 단순한 질문에 인류는 수천 년째 답을 찾고 있다.
```

## AP-07: 독자를 가르치는 어조

**기준**: 글쓴이와 독자가 같은 자리에 있는가, 위아래 관계인가.

**수정 대상 예시**:
```
현대 시민이라면 경제의 기본 원리를 반드시 알아야 한다.
```

**수정 후**:
```
경제를 어렵게 느끼는 사람이 많다. 하지만 경제는 숫자의 학문이 아니다.
사람이 어떻게 선택하는가에 대한 이야기다. 이렇게 보면, 우리는 이미 경제를 매일 경험하고 있다.
```

**허용 예시**: 독자의 생각을 대변하는 문장은 가르치는 것이 아니다.
```
당신은 아마 이렇게 생각할 것이다. 그것은 나와 상관없는 이야기라고.
```
→ 독자 시점을 대변한다. 허용한다.

---

## AP-08: 깊이 없는 마무리

**기준**: 통찰이 과정에서 나왔는가, 선언됐는가.
글쓴이가 "이것이 바로 통찰입니다"라고 알려주는 구조는 피한다.
독자가 스스로 도달했다고 느껴야 한다.

**수정 대상 예시**:
```
자유란 무엇인가? 그것은 자신의 삶을 스스로 선택하는 것이다.
```
→ 도입 질문에서 바로 결론으로 건너뛴다. 과정이 없다.

**수정 방향**: 도입 → 맥락 → 설명 → 연결 이후에 통찰이 와야 한다.
`SKILL.md`의 글의 기본 구조 참고.

## 빠른 검수 체크리스트

```
□ 감성 언어 앞에 구조적 설명이 쌓여 있는가 (AP-01)
□ 독자에게 행동을 명령·촉구하고 있지 않은가 (AP-02)
□ 결론 앞에 맥락과 설명이 충분한가 (AP-03)
□ 전문 용어에 일상어 번역이 병행됐는가 (AP-04)
□ 두 입장 나열 후 구조적 차이를 드러냈는가 (AP-05)
□ 수동형 정의 나열 대신 능동적 서술인가 (AP-06)
□ 글쓴이와 독자가 같은 자리에 있는가 (AP-07)
□ 통찰이 과정에서 나왔는가, 선언됐는가 (AP-08)
```
