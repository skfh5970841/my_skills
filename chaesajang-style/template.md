# template.md — 채사장 스타일 글쓰기 예시

이 파일은 공식이 아니다. 좋은 예시와 그것이 작동하는 이유다.
예시를 복사하지 말고, 왜 작동하는지를 파악해서 주제에 맞게 판단한다.

---

## 도입: 어떻게 시작할 것인가

채사장의 도입은 하나의 목표만 있다. 독자를 질문 앞에 세우는 것.
그 방법이 세 가지일 뿐이다. 레이블이 아니라 작동 원리를 본다.

---

### 질문형 도입

독자가 이미 하고 있는 행위 안에 숨어 있는 질문을 꺼낸다.
질문이 독자 자신의 것이어야 작동한다.
"세금이란 무엇인가"는 세금을 내는 사람의 질문이다.
"헤겔의 변증법이란 무엇인가"는 아직 독자의 질문이 아니다.

```
우리는 매달 세금을 낸다.
그런데 '세금이 무엇인가'라는 질문에 제대로 답할 수 있는 사람은 많지 않다.
```

```
살면서 한 번쯤 '내 노력이 왜 결과로 이어지지 않을까'를 생각해본 적 있을 것이다.
이 질문은 보통 자신을 향한다.
그런데 방향을 바꿔보면 어떨까.
```

```
역사는 반복된다.
이 말이 위안인지 경고인지는, 역사를 어떻게 읽느냐에 달려 있다.
```

세 예시의 구조는 다르다. 공통점은 하나다. 독자를 질문 앞에 세운다.

---

### 역설형 도입

독자가 당연하게 여기는 통념을 먼저 인정한 뒤 뒤집는다.
반전이 강할수록 좋다. 단, 뒤집을 통념이 독자 대부분이 실제로 믿는 것이어야 한다.

```
우리는 민주주의가 다수결이라고 배웠다.
그런데 다수결은 민주주의의 여러 요소 중 하나일 뿐이다.
이 차이가 작아 보이지만, 여기서 민주주의 이해가 완전히 달라진다.
```

```
과학은 진리를 발견한다고 우리는 배웠다.
그런데 과학자들은 진리를 발견하는 것이 아니라, 더 나은 설명을 선택하는 것이라고 말한다.
```

---

### 역사적 장면형 도입

구체적인 역사 장면으로 바로 시작한다.
첫 문장이 평서문이어도 괜찮다. 장면 자체가 진입점이다.
현재와 연결된다는 암시를 마지막에 둔다. 연결을 직접 설명하지 않는다.

```
1929년, 뉴욕 월스트리트에서 주가가 폭락했다.
단 며칠 만에 수백만 명의 삶이 무너졌다.
이 사건은 경제 위기가 아니었다. 세계가 작동하는 방식에 대한 질문이었다.
```

```
1517년, 마르틴 루터는 교회 문에 95개의 반박문을 붙였다.
그는 교회를 무너뜨리려 한 것이 아니었다.
그러나 그 행동이 유럽을 반으로 갈랐다.
```

이 방식이 질문형보다 강한 진입점일 때를 선택한다.

## 개념 설명: 추상에서 구체로

기본 방향은 추상 → 구체 → 비유 → 한 문장 정리다.
비유는 개념을 더 명확하게 만들 때만 추가한다. 비유가 목적이 아니다.

```
잉여 가치란 노동자가 만들어낸 가치 중 임금으로 돌아오지 않는 부분이다.
하루에 100만 원어치 제품을 만드는 노동자가 30만 원을 받는다면,
나머지 70만 원은 자본가에게 귀속된다.
마르크스는 이 구조가 자본주의의 우연한 결함이 아니라 작동 원리 자체라고 봤다.
쉽게 말하면, 내가 일한 시간 중 나를 위한 시간과 타인을 위한 시간의 차이다.
결국 잉여 가치는 자본주의가 유지되는 연료다.
```

독자가 이미 개념을 어느 정도 알고 있다면, 아는 설명을 먼저 인정하고 더 깊이 들어간다.

```
자유란 하고 싶은 것을 할 수 있는 상태라고 흔히 말한다.
사실 이것은 자유의 절반만 담은 설명이다.
더 정확하게 말하면, 자유는 하기 싫은 것을 거부할 수 있는 상태다.
이 차이는 우리가 어떤 사회 제도를 설계하느냐에서 결정적으로 드러난다.
```

## 전개: 예상치 못한 연결

하나의 개념을 설명한 뒤, 전혀 다른 분야와의 연결로 독자가 예상하지 못한 시야를 만든다.
연결은 억지로 만들지 않는다. 실제로 구조가 같을 때만 연결한다.

```
마르크스는 경제가 역사를 만든다고 했다.
다윈은 환경이 생명을 만든다고 했다.
프로이트는 무의식이 인간을 만든다고 했다.
셋 다 같은 말을 하고 있다. 우리가 의식하지 못하는 무언가가 우리를 움직인다는 것이다.
```

같은 구조의 문장을 반복할 때, 반복 횟수는 리듬이 완성된다고 느껴지는 지점에서 결정한다.
2회로 충분하다면 굳이 3회를 채우지 않는다.

## 통찰: 어떻게 마무리할 것인가

마무리의 방향은 세 가지다.

**도입으로 귀환**: 처음 질문을 다시 꺼내, 본문에서 쌓은 맥락으로 다른 답을 제시한다.
```
처음 질문으로 돌아가보자. 세금이란 무엇인가.
이제 이 질문은 납세의 의무에 관한 것이 아니다.
우리가 어떤 사회에서 살고 싶은가에 관한 질문이다.
```

**인식의 역전**: 독자가 갖고 있던 인식을 명시적으로 뒤집는다.
```
우리는 경제를 숫자의 학문이라고 생각해왔다.
그런데 경제는 인간의 욕망과 공포가 만들어내는 이야기다.
```

**확장 후 착지**: 개인 → 사회 → 역사로 넓혔다가, 마지막 문장을 독자의 현재로 돌린다.

셋 중 무엇을 고르든, 글쓴이가 통찰을 선언하지 않는다.
독자가 스스로 도달한 것처럼 느껴야 한다.

## 종합 예시

**주제: 세금이란 무엇인가**

```
우리는 매달 세금을 낸다.
그런데 '세금이 무엇인가'라는 질문에 제대로 답할 수 있는 사람은 많지 않다.

세금은 단순히 국가가 가져가는 돈이 아니다.
우리가 사회를 유지하기 위해 합의한 비용이다.
문제는, 그 합의가 언제 어떻게 이루어졌는지를 기억하는 사람이 없다는 것이다.

역사적으로 세금은 권력의 언어였다.
왕은 세금을 통해 자신이 누구의 위에 있는지를 증명했다.
그런데 근대 이후 민주주의가 확산되면서 세금의 의미가 달라지기 시작했다.
권력의 언어에서 시민의 언어로.

세금을 '빼앗기는 것'으로 보는 사람은 사회를 개인들의 집합으로 본다.
세금을 '함께 내는 비용'으로 보는 사람은 사회를 공동체로 본다.
이 두 시각 사이의 갈등이 현대 정치의 본질이다.

처음 질문으로 돌아가보자. 세금이란 무엇인가.
이제 이 질문은 납세의 의무에 관한 것이 아니다.
우리가 어떤 사회에서 살고 싶은가에 관한 질문이다.
```
