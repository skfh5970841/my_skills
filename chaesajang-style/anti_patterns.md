# anti_patterns.md — 채사장 스타일 흔한 함정

이 파일은 금지 목록이 아니다.
채사장 스타일을 흉내낼 때 자주 빠지는 함정과, 그 함정을 알아보는 기준이다.
어떤 표현이 문제가 아니라, 그 표현이 글 안에서 어떤 역할을 하는지가 기준이다.

---

## 감성이 논리를 대체하는 경우

**알아보는 기준**: 감성 문장을 삭제했을 때 앞뒤가 자연스럽게 연결되는가.
연결된다면, 그 문장은 설명 없이 감성으로 결론을 낸 것이다.

```
역사는 우리에게 가슴 아픈 교훈을 남긴다.
수많은 사람들의 눈물과 희생 위에 지금 우리가 서 있다는 것을 잊지 말아야 한다.
```
→ 구조적 설명 없이 감성으로 결론을 낸다.

```
[앞에 구조적 설명이 충분히 쌓인 상태에서]
같은 전쟁이 같은 이유로 반복된 것이다. 희생자들의 이름이 달라졌을 뿐.
```
→ 마지막 문장의 감성은 앞의 논리에서 나온다. 이것은 함정이 아니다.

---

## 자기계발 클리셰

**알아보는 기준**: 독자에게 행동이나 태도를 명령·촉구하는가.
채사장의 글은 독자를 변화시키려 하지 않는다. 독자 스스로 생각하게 만든다.

```
지식은 당신의 삶을 바꿀 수 있다. 지금 당장 책을 펼쳐라. 아는 것이 힘이다.
```
→ 독자에게 명령한다.

```
이 구조를 이해하면, 지금 일어나는 일들이 다르게 보일 수 있다.
```
→ 가능성을 보여준다. 명령이 아니다.

---

## 근거 없는 단언

**알아보는 기준**: 결론 앞에 독자가 납득할 수 있는 맥락이 충분히 쌓였는가.
쌓였다면 강한 결론 문장도 힘이 된다. 쌓이지 않았다면 단언이다.

```
결국 자본주의는 실패한 시스템이다.
```
→ 맥락 없이 단언한다.

```
[앞에 역사적 사례와 구조 설명이 충분히 쌓인 상태에서]
결국 이것은 제도의 실패가 아니라 설계의 문제였다.
```
→ 같은 단언 구조지만, 앞에서 충분히 쌓였다. 함정이 아니다.

---

## 전문 용어 번역 없이 나열

**알아보는 기준**: 독자가 그 용어를 모를 가능성이 있는가.
독자 맥락상 이미 알고 있다면 번역 없이 써도 된다.

```
포스트모더니즘은 메타서사의 해체와 다원주의적 인식론을 기반으로 하며,
계몽주의적 이성 중심주의에 대한 비판적 전복을 시도한다.
```
→ 번역 없이 용어를 나열한다.

```
포스트모더니즘은 간단히 말해, '절대적 진리는 없다'는 선언이다.
계몽주의가 이성으로 하나의 올바른 답을 찾을 수 있다고 믿었다면,
포스트모더니즘은 그 믿음 자체를 의심한다.
```
→ 용어를 일상어로 번역한다.

---

## 판단 포기 양비론

**알아보는 기준**: 두 입장을 나열한 뒤 아무 판단도 하지 않는가.
중립이 목표가 아니다. 두 입장이 추구하는 가치의 차이를 드러내는 것이 목표다.

```
자본주의에도 장단점이 있고, 사회주의에도 장단점이 있다.
어느 쪽이 더 좋은지는 사람마다 다르다.
```
→ 차이를 드러내지 않고 판단을 회피한다.

```
자본주의와 사회주의는 서로 다른 가치를 극대화하려 한다. 자본주의는 효율을, 사회주의는 평등을.
문제는 어느 쪽이 더 나은가가 아니라, 우리가 어떤 가치를 더 중요하게 여기는가다.
```
→ 판단을 회피하지 않고 구조적 차이를 드러낸다.

---

## 논문·교과서체

**알아보는 기준**: 문장이 능동적으로 움직이는가, 수동적으로 정의되는가.

```
경제는 재화와 서비스의 생산, 분배, 소비에 관한 활동의 총체로 정의될 수 있으며,
이는 미시경제학과 거시경제학의 두 분야로 연구된다.
```
→ 정의를 나열한다. 방향이 없다.

```
경제는 우리가 어떻게 먹고사는가에 대한 학문이다. 거창하게 들리지만 질문은 하나다.
누가 무엇을 얼마나 갖는가. 이 단순한 질문에 인류는 수천 년째 답을 찾고 있다.
```
→ 문장이 움직인다.

---

## 독자를 가르치는 어조

**알아보는 기준**: 글쓴이와 독자가 같은 자리에 있는가.

```
현대 시민이라면 경제의 기본 원리를 반드시 알아야 한다.
```
→ 위에서 내려다본다.

```
경제를 어렵게 느끼는 사람이 많다. 하지만 경제는 숫자의 학문이 아니다.
사람이 어떻게 선택하는가에 대한 이야기다.
```
→ 같은 자리에서 같은 방향을 본다.

독자의 생각을 대변하는 문장은 가르치는 것이 아니다.
```
당신은 아마 이렇게 생각할 것이다. 그것은 나와 상관없는 이야기라고.
```
→ 독자 시점을 대변한다. 함정이 아니다.

---

## 깊이 없는 마무리

**알아보는 기준**: 통찰이 과정에서 나왔는가, 선언됐는가.

```
자유란 무엇인가? 그것은 자신의 삶을 스스로 선택하는 것이다.
```
→ 도입 질문에서 바로 결론으로 건너뛴다. 과정이 없다.

통찰은 독자가 스스로 도달한 것처럼 느껴야 한다.
"이것이 바로 ~"처럼 글쓴이가 통찰을 선언하면, 독자는 수동적이 된다.

---

## 아류작이 되는 네 가지 패턴

채사장 스타일을 모방한 글이 "흉내"로 느껴지는 이유는 네 가지 중 하나다.

**연결 없는 나열.**
각 분야를 따로 설명하고 순서대로 배치하는 구조.
채사장의 핵심은 개별 지식이 아니라 지식 사이의 연결선이다.
"정치 편, 경제 편, 철학 편"으로 나누는 순간 이미 채사장이 아니다.

**무색무취한 중립.**
채사장은 중립을 표방하지만 뚜렷한 방향성을 갖는다.
완전한 중립으로 모방하면 글의 에너지가 사라진다.
"A도 맞고 B도 맞다"는 판단 회피다. "A와 B는 이런 구조적 차이가 있다"가 채사장이다.

**삽화로 쓰는 비유.**
채사장의 비유는 장식이 아니라 구조적 등가물이다.
비유를 읽은 뒤 독자가 개념을 더 잘 이해했다면 기능한 것이다.
비유 자체는 아름답지만 개념과 따로 논다면 삽화다.
흉내내는 글은 비유를 삽화로 쓰고, 채사장은 비유를 논증의 대체물로 쓴다.

**겸손한 불확신.**
채사장의 대화체는 겸손이 아니라 편안한 확신이다.
"저는 이렇게 생각합니다"가 아니라 "이건 이런 거야"다.
"~일 수 있습니다", "~라고 볼 수도 있겠지만"으로 가득한 글은 채사장이 아니다.
