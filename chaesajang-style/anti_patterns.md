# anti_patterns.md — 채사장 스타일 흔한 함정

이 파일은 금지 목록이 아니다.
채사장 스타일을 흉내낼 때 자주 빠지는 함정과, 그 함정을 알아보는 기준이다.
어떤 표현이 문제가 아니라, 그 표현이 글 안에서 어떤 역할을 하는지가 기준이다.

---

## 감성이 논리를 대체하는 경우

**알아보는 기준**: 감성 문장을 삭제했을 때 앞뒤가 자연스럽게 연결되는가.
연결된다면, 그 문장은 설명 없이 감성으로 결론을 낸 것이다.

```
역사는 우리에게 가슴 아픈 교훈을 남긴다.
수많은 사람들의 눈물과 희생 위에 지금 우리가 서 있다는 것을 잊지 말아야 한다.
```
→ 구조적 설명 없이 감성으로 결론을 낸다.

```
[앞에 구조적 설명이 충분히 쌓인 상태에서]
같은 전쟁이 같은 이유로 반복된 것이다. 희생자들의 이름이 달라졌을 뿐.
```
→ 마지막 문장의 감성은 앞의 논리에서 나온다. 이것은 함정이 아니다.

---

## 자기계발 클리셰

**알아보는 기준**: 독자에게 행동이나 태도를 명령·촉구하는가.
채사장의 글은 독자를 변화시키려 하지 않는다. 독자 스스로 생각하게 만든다.

```
지식은 당신의 삶을 바꿀 수 있다. 지금 당장 책을 펼쳐라. 아는 것이 힘이다.
```
→ 독자에게 명령한다.

```
이 구조를 이해하면, 지금 일어나는 일들이 다르게 보일 수 있다.
```
→ 가능성을 보여준다. 명령이 아니다.

---

## 근거 없는 단언

**알아보는 기준**: 결론 앞에 독자가 납득할 수 있는 맥락이 충분히 쌓였는가.
쌓였다면 강한 결론 문장도 힘이 된다. 쌓이지 않았다면 단언이다.

```
결국 자본주의는 실패한 시스템이다.
```
→ 맥락 없이 단언한다.

```
[앞에 역사적 사례와 구조 설명이 충분히 쌓인 상태에서]
결국 이것은 제도의 실패가 아니라 설계의 문제였다.
```
→ 같은 단언 구조지만, 앞에서 충분히 쌓였다. 함정이 아니다.

---

## 전문 용어 번역 없이 나열

**알아보는 기준**: 독자가 그 용어를 모를 가능성이 있는가.
독자 맥락상 이미 알고 있다면 번역 없이 써도 된다.

```
포스트모더니즘은 메타서사의 해체와 다원주의적 인식론을 기반으로 하며,
계몽주의적 이성 중심주의에 대한 비판적 전복을 시도한다.
```
→ 번역 없이 용어를 나열한다.

```
포스트모더니즘은 간단히 말해, '절대적 진리는 없다'는 선언이다.
계몽주의가 이성으로 하나의 올바른 답을 찾을 수 있다고 믿었다면,
포스트모더니즘은 그 믿음 자체를 의심한다.
```
→ 용어를 일상어로 번역한다.

---

## 판단 포기 양비론

**알아보는 기준**: 두 입장을 나열한 뒤 아무 판단도 하지 않는가.
중립이 목표가 아니다. 두 입장이 추구하는 가치의 차이를 드러내는 것이 목표다.

```
자본주의에도 장단점이 있고, 사회주의에도 장단점이 있다.
어느 쪽이 더 좋은지는 사람마다 다르다.
```
→ 차이를 드러내지 않고 판단을 회피한다.

```
자본주의와 사회주의는 서로 다른 가치를 극대화하려 한다. 자본주의는 효율을, 사회주의는 평등을.
문제는 어느 쪽이 더 나은가가 아니라, 우리가 어떤 가치를 더 중요하게 여기는가다.
```
→ 판단을 회피하지 않고 구조적 차이를 드러낸다.

판단의 위임과 구별: 두 구조를 공정하게 제시한 뒤 선택을 독자에게 넘기는 것은 판단 회피가 아니다. 단, 각 입장이 추구하는 가치의 차이가 명확히 드러난 뒤에만 가능하다. 차이를 드러내지 않고 넘기면 양비론이 된다. → `template.md`의 "판단의 위임" 참고.

---

## 논문·교과서체

**알아보는 기준**: 문장이 능동적으로 움직이는가, 수동적으로 정의되는가.

```
경제는 재화와 서비스의 생산, 분배, 소비에 관한 활동의 총체로 정의될 수 있으며,
이는 미시경제학과 거시경제학의 두 분야로 연구된다.
```
→ 정의를 나열한다. 방향이 없다.

```
경제는 우리가 어떻게 먹고사는가에 대한 학문이다. 거창하게 들리지만 질문은 하나다.
누가 무엇을 얼마나 갖는가. 이 단순한 질문에 인류는 수천 년째 답을 찾고 있다.
```
→ 문장이 움직인다.

---

## 독자를 가르치는 어조

**알아보는 기준**: 글쓴이와 독자가 같은 자리에 있는가.

```
현대 시민이라면 경제의 기본 원리를 반드시 알아야 한다.
```
→ 위에서 내려다본다.

```
경제를 어렵게 느끼는 사람이 많다. 하지만 경제는 숫자의 학문이 아니다.
사람이 어떻게 선택하는가에 대한 이야기다.
```
→ 같은 자리에서 같은 방향을 본다.

독자의 생각을 대변하는 문장은 가르치는 것이 아니다.
```
당신은 아마 이렇게 생각할 것이다. 그것은 나와 상관없는 이야기라고.
```
→ 독자 시점을 대변한다. 함정이 아니다.

---

## 깊이 없는 마무리

**알아보는 기준**: 통찰이 과정에서 나왔는가, 선언됐는가.

```
자유란 무엇인가? 그것은 자신의 삶을 스스로 선택하는 것이다.
```
→ 도입 질문에서 바로 결론으로 건너뛴다. 과정이 없다.

통찰은 독자가 스스로 도달한 것처럼 느껴야 한다.
"이것이 바로 ~"처럼 글쓴이가 통찰을 선언하면, 독자는 수동적이 된다.

---

## 아류작이 되는 네 가지 패턴

채사장 스타일을 모방한 글이 "흉내"로 느껴지는 이유는 네 가지 중 하나다.

**연결 없는 나열.**
각 분야를 따로 설명하고 순서대로 배치하는 구조.
채사장의 핵심은 개별 지식이 아니라 지식 사이의 연결선이다.
"정치 편, 경제 편, 철학 편"으로 나누는 순간 이미 채사장이 아니다.

**무색무취한 중립.**
채사장은 중립을 표방하지만 뚜렷한 방향성을 갖는다.
완전한 중립으로 모방하면 글의 에너지가 사라진다.
"A도 맞고 B도 맞다"는 판단 회피다. "A와 B는 이런 구조적 차이가 있다"가 채사장이다.

**삽화로 쓰는 비유.**
채사장의 비유는 장식이 아니라 구조적 등가물이다.
비유를 읽은 뒤 독자가 개념을 더 잘 이해했다면 기능한 것이다.
비유 자체는 아름답지만 개념과 따로 논다면 삽화다.
흉내내는 글은 비유를 삽화로 쓰고, 채사장은 비유를 논증의 대체물로 쓴다.

**겸손한 불확신.**
채사장의 대화체는 겸손이 아니라 편안한 확신이다.
"저는 이렇게 생각합니다"가 아니라 "이건 이런 거야"다.
"~일 수 있습니다", "~라고 볼 수도 있겠지만"으로 가득한 글은 채사장이 아니다.

---

## 클로드가 흔히 빠지는 네 가지 함정

이 스킬은 클로드가 실행한다. 클로드의 글쓰기 기본값은 채사장 스타일과 자주 충돌한다.
앞 절의 네 가지가 일반적인 모방자의 함정이라면, 아래 네 가지는 클로드에게 특히 자주 발생하는 함정이다.
이 네 가지에 빠지지 않았는지를 출력 직전에 반드시 점검한다.

---

### Hedging 마비

**알아보는 기준**: 단정해야 할 자리에 가능성·추측 표현이 연쇄적으로 등장하는가.
채사장의 "편안한 확신"은 단정에서 나온다. 한 단락 안에 "~일 수도 있다", "~라고 볼 수 있다", "어쩌면"이 두 번 이상 나오면 글이 자기 무게를 잃는다.

```
이 구조가 자본주의의 핵심이라고 볼 수 있을 것이다.
어쩌면 이것이 우리가 놓치고 있는 부분일 수도 있다.
```
→ 단정해야 할 자리에서 두 번 연속 도망친다. 확신이 침식된다.

```
이 구조가 자본주의의 핵심이다.
우리는 이것을 놓치고 있다.
```
→ 같은 내용을 단정한다. 단정이 위압이 아니라 초대로 작동하려면, 앞에서 맥락이 충분히 쌓여 있어야 한다는 점만 지킨다.

판단 기준: 가능성 표현은 진짜로 불확실할 때만 쓴다. 확신을 부드럽게 만들기 위한 완충재가 아니다.

---

### 불릿·번호 매기기로 도주

**알아보는 기준**: 산문이 흘러야 할 자리에 항목 리스트가 들어와 있는가.
채사장의 글은 산문이다. 리스트는 항목 사이의 인과·관계를 끊고, 연결이 끊긴 순간 채사장이 아니다.

```
봉건제의 특징은 다음과 같다.
- 토지가 권력의 기반이었다
- 신분 제도가 작동했다
- 종교가 권위를 정당화했다
```
→ 세 항목의 관계가 보이지 않는다. 단순 나열로 끝난다.

```
봉건제는 토지에서 출발한다. 땅을 가진 사람이 권력을 가졌고,
그 권력을 정당화하기 위해 신분 제도가 필요했다.
그 신분 제도를 신성하게 만들기 위해 종교가 동원됐다.
```
→ 세 요소가 서로를 지탱하는 하나의 구조로 묶인다.

예외는 있다. 진짜로 병렬이고 순서가 없는 항목, 절차 안내, 참고 자료 목록 같은 자리에서는 리스트가 정당하다. 그러나 본문 전개에서는 거의 항상 산문이 옳다.

---

### 결론 회피로 끝맺기

**알아보는 기준**: 글의 마지막 문장이 통찰을 던지는가, 통찰을 미루는가.
"이는 흥미로운 질문을 제기한다", "다양한 관점이 가능하다", "앞으로 더 많은 논의가 필요하다" — 이것들은 결론의 자리에 결론 미루기를 놓은 것이다.

```
이상에서 살펴본 바와 같이, 자유의 개념은 복잡하며,
이에 대한 다양한 시각이 존재한다. 이는 앞으로도 깊이 있는 논의를 요하는 주제다.
```
→ 글의 끝에서 글이 시작점으로 후퇴한다. 본문에서 쌓은 것이 흩어진다.

```
자유는 하고 싶은 것을 할 수 있는 능력이 아니다.
하기 싫은 것을 거부할 수 있는 능력이다.
이 차이가 우리가 어떤 사회를 만들 것인가를 결정한다.
```
→ 본문이 쌓아온 것을 한 점으로 모은다. 결론은 도착이지 출발이 아니다.

균형 강박과 구별: 두 입장을 나열하고 판단을 회피하는 것은 "판단 포기 양비론"(위 참고)이다. 결론 회피는 단일 주제에서도 발생한다. 글쓴이가 "결론은 독자가 정할 일"이라는 태도로 도망친 결과다.

---

### 메타 설명

**알아보는 기준**: 글이 글 자체를 가리키는 문장이 있는가.
"이 글에서는 ~을 다루겠다", "지금까지 살펴본 바와 같이", "이어서 ~을 살펴보자" — 이런 문장은 글 바깥에서 글을 안내하는 가이드의 목소리다.

채사장의 글에는 안내자가 없다. 글이 곧 길이고, 독자는 그 위를 걸을 뿐이다.

```
이 글에서는 먼저 자본주의의 정의를 살펴본 뒤,
그것이 현대 사회에 미친 영향을 분석하고, 마지막으로 대안을 모색해보고자 한다.
```
→ 글이 시작도 하기 전에 글을 요약한다. 독자가 이미 길의 끝을 본다.

```
우리는 매일 물건을 사고판다.
그런데 한 번이라도 '왜 이것이 내 것인가'를 생각해본 적이 있는가.
```
→ 안내 없이 바로 질문 앞에 독자를 세운다.

예외: 매우 긴 글에서 독자가 길을 잃을 위험이 있을 때 한 번의 메타 신호("앞에서 본 것을 정리하면" 정도)는 허용된다. 그러나 두 번 이상 등장하면 글의 무게가 사라진다.
